Sci-Hub Scholar Options

Cache Management

0

Clearing cached DOIs will remove locally stored article-to-DOI mappings. The extension will re-fetch this information as needed. Configuration settings (like custom URL) will not be cleared.

Custom Sci-Hub Domain

Specify a custom Sci-Hub domain (e.g., https://sci-hub.st). This URL will be used to access articles. If left blank, the extension will attempt to use an automatically discovered or a default URL.
